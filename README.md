# Open Weather API

- this project is built to consume Open Weather API and display the recieved data.

- **NO UI framework** is used, Except for the loading indicator I used Bootstrap

- **self-explanatory comments** are used
- **error messages** are displayed when the api returns an error
- **Redux** is used to handle app state
- **SASS** is used
- **React Hooks** are used
- It's **Responsive**
